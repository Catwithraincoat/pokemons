# Pokemons
Проект создан для демонстрации способности написания ООП кода на чистой Java.

## О проекте

В проекте испольузются сторонний архив Pokemon.jar. От классов внутри него _(ex. Pokemon)_ наследуются основные сущности-покемоны, действия покемонов и Битва.

## Запуск
Запуск через конечный jar  архив или через функцию main.
