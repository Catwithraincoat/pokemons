# Pokemons
Проект создан для демонстрации способности написания ООП кода на чистой Java.

## О проекте

В проекте испольузются сторонний архив Pokemon.jar. От классов внутри него _(ex. Pokemon)_ наследуются основные сущности-покемоны, атаки покемонов и сущность Битва.

### Цель
Создать две команды покемонов и поставить их на поле боя для битвы.
